# Stranger Things: An Analysis of Character, Location, and Sentiment 

*Stranger Things* has been a pop culture phenomenon since it originally aired in 2016. The first half of the latest and final season dropped on November 26, 2025. The show has an intricate network of characters and locations whose interactions can be studied to understand any patterns among appearances of characters.
